**GitHub AST** (also known as **AI-Scaler-Toolkit**) is a unified, cross-platform LLM inference and training server built on **FastAPI**. It abstracts hardware complexity (CUDA vs. Intel XPU vs. CPU) and supports dynamic model management, enabling users to run large language models on consumer-grade hardware with varying acceleration capabilities.
