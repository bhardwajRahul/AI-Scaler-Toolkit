# Trusta AST LLM Wiki

Bilingual knowledge base for the **Trusta AST (AI Scaler Toolkit)** LLM inference/training service, plus the related **TRUSTA enterprise SSD** product line and the LLM-Wiki methodology used to build it.

雙語知識庫：涵蓋 **Trusta AST（AI Scaler Toolkit）** LLM 推理／訓練服務、相關的 **TRUSTA 企業級 SSD** 產品線，以及建置本 wiki 所用的 LLM Wiki 方法論。

## Languages / 語言

- 🇬🇧 English — [`en/index.md`](en/index.md)
- 🇹🇼 繁體中文 — [`zh/index.md`](zh/index.md)

Each language folder mirrors the same page tree. `raw/` holds the original ingested sources (HTML/Markdown) and is shared, language-agnostic, and not translated.

兩個語言資料夾具有相同的頁面結構。`raw/` 存放原始匯入來源（HTML/Markdown），為共用、不分語言、不翻譯。

## Structure / 結構

```
wiki/
├─ en/            # English pages
├─ zh/            # 繁體中文頁面
└─ raw/           # original ingested sources (shared, not translated)
```
