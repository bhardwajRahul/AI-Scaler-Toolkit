**GitHub AST**（也稱為 **AI-Scaler-Toolkit**）是一個建構於 **FastAPI** 之上的統一、跨平台 LLM 推論與訓練伺服器。它抽象化了硬體的複雜性（CUDA 與 Intel XPU 與 CPU），並支援動態模型管理，讓使用者能夠在具備各種加速能力的消費級硬體上執行大型語言模型。
